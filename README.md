# Bilal Ahmad — Portfolio

A modern, premium, responsive personal portfolio website for a Full Stack MERN Developer.

## Tech Stack

- **React.js** (Vite)
- **Tailwind CSS** with dark/light mode
- **Framer Motion** animations
- **React Icons**

## Features

- Hero, About, Skills, Services, Projects, Experience Timeline, Testimonials, Contact, Footer
- Dark mode toggle (persisted in localStorage)
- Glassmorphism, modern gradients, smooth scrolling
- Mobile-first responsive design, SEO meta tags

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

## Customize

- Edit content in `src/data/portfolio.js` (skills, projects, socials, testimonials, timeline)
- Replace `` with your real resume
- Update social/contact links in `src/data/portfolio.js`
